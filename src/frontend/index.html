REFERENCIAS PARA TESTEAR PROYECTO EN POSTMAN

LOGIN

Credenciales de Prueba: { "email": "test@mail.com", "password": "1234"}

en BODY copiar el objeto con las credenciales

POST: http://localhost:3000/usuarios/login

agregar en Header: Key=Authorization Value: Bearer TOKEN GENERADO para continuar el test

LISTADO DE USUARIOS

GET: http://localhost:3000/usuarios

LISTADO DE INMUEBLES

GET: http://localhost:3000/inmuebles

MOSTRAR UN INMUEBLE SEGUN SU ID

GET: http://localhost:3000/inmuebles/:id

copiar un id de un inmueble del listado y pegarlo en lugar :id en la URL

AGREGAR UN NUEVO INMUEBLE

POST: http://localhost:3000/inmuebles/:id

en BODY ingresar como objeto los datos del inmueble, ejemplo:

{"Tipo ": "Casa", "Domicilio": "Av. San Pedrito 200", "Precio": 50000, "Ambientes": 2}

MODIFICAR UNO o TODOS LOS CAMPOS DE UN INMUEBLE

PUT: http://localhost:3000/inmuebles/:id

copiar un id de un inmueble del listado y pegarlo en lugar :id en la URL

ELIMINAR UN INMUEBLE

DELETE: http://localhost:3000/inmuebles/:id

copiar un id de un inmueble del listado y pegarlo en lugar :id en la URL
